⚠️ **Important**  
Anonomi should only be downloaded from **official sources** listed on this page.  
Do **not** trust third-party mirrors, re-packaged APKs, or links shared in chats.

---

## Android

### GitHub Releases (recommended)
The primary distribution channel for Anonomi is GitHub Releases.

- Official repository: https://github.com/anonomi-org/anonomi-messenger
- Releases page: https://github.com/anonomi-org/anonomi-messenger/releases

Each release includes:
- APK file
- Version number
- Changelog
- Cryptographic fingerprints or signatures (when applicable)

This is the **authoritative source**.

---

## Verifying authenticity

Before installing Anonomi, you should verify that the file is authentic.

Recommended steps:
1. Download the APK from an official source
2. Compare the published fingerprint or signature
3. Ensure it matches the one documented in the release notes

See:
- [Security & verification](/docs/security)

Verification is especially important if you:
- Operate in a hostile network
- Are targeted by surveillance or malware
- Received the download link from a third party

---

## Updates

Anonomi does **not** use automatic background updates from centralized servers.

Updates are released deliberately and announced through official channels.  
You are responsible for checking and installing updates.

---

## What *not* to do

- ❌ Do not download APKs from random websites
- ❌ Do not trust re-hosted binaries
- ❌ Do not install modified builds unless you audited the source

If in doubt, **do not install**.

---

## Questions or doubts

If you are unsure whether a download is legitimate:
- Stop
- Verify against official sources
- Ask through the official project channels listed on the website
